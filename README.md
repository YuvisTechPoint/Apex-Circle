# Apex Circle — React Website

A complete multi-page React website for Apex Circle built with Vite.

## Pages Included
- Home (Hero, Partners, About, Programs, Hackathons, Testimonials, FAQ)
- Hackathons (Hero, Prizes, Timeline, Top Projects)
- Events (Workshops & Meetups)
- Projects (Showcase with filters)
- Team (Leadership + Departments)
- Gallery (Filterable grid)
- Community / Join (6 paths + application form)
- About (Story, Mission, Vision, Values)
- Contact (Info + contact form)

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Customization
- Replace placeholder stats in `src/App.jsx`
- Add real images to the Gallery section
- Wire forms to a backend (Formspree, Firebase, etc.)
- Update partner/sponsor logos
- For multi-page routing, swap the internal router with `react-router-dom`

## Tech Stack
- React 18
- Vite 5
- Pure inline styles (no CSS framework needed)
- Google Fonts: Space Mono + Syne
